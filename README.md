# My-JARVIS
I wanted to build an Agentic Ai for the first time (JARVIS) inspired from Ironman movie not gonna lie . "For you, sir, always."

## 🚀 Features

* 🎤 Voice Recognition (Speech-to-Text)
* 🔊 Text-to-Speech response
* 📂 Open applications
* 📄 Read text from files
* 🧠 Basic command processing

## 🛠️ Tech Stack

* Python
* pyttsx3
* SpeechRecognition

## 📂 Project Structure

```
actions/   → Handles app & file operations  
brain/     → Command processing / NLP  
voice/     → Speech input & output  
main.py    → Main program file  
```

## ▶️ How to Run

1. Clone the repository
2. Install dependencies:

```
pip install -r requirements.txt
```

3. Run the assistant:

```
python main.py
```
